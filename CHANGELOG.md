# Changelog

## [0.1.0] - 2026-06-21

### Added

- Persian RTL web interface with dark and light themes
- Reverse TCP multiplexing
- Reverse WSS multiplexing with certificate verification
- Multi-port tunnel mappings
- Encrypted tunnel-secret storage
- Secure panel authentication and CSRF protection
- Live metrics, process logs, backups, and autostart
- Hardened systemd installation
- AMD64 and ARM64 release builds
